| [AdminBrandServiceImpl.java#L45-49](d:\Java_study\shop_v3\GoBuy\src\main\java\com\example\gobuy\modules\admin\service\impl\AdminBrandServiceImpl.java#L45-49) | `listBrands()` 在 forEach 循环内调用 `countProducts(brand.getId())`，每个品牌执行一次 SQL | 100个品牌 = 101次SQL |

| [RoleServiceImpl.java#L60-65](d:\Java_study\shop_v3\GoBuy\src\main\java\com\example\gobuy\modules\admin\service\impl\RoleServiceImpl.java#L60-65) | `listRoles()` 对每个角色执行 `adminRoleMapper.selectCount()` | 100个角色 = 101次SQL |

**修复方案**：使用批量查询 + 内存组装

```java
// 修复示例：AdminBrandServiceImpl.listBrands
@Override
public IPage<BrandVO> listBrands(BrandQueryDTO queryDTO) {
    // 先查分页品牌
    IPage<Brand> page = new Page<>(queryDTO.getPage(), queryDTO.getSize());
    LambdaQueryWrapper<Brand> wrapper = buildWrapper(queryDTO);
    IPage<Brand> result = page(page, wrapper);

    // 批量查询商品数量
    List<Long> brandIds = result.getRecords().stream().map(Brand::getId).collect(Collectors.toList());
    Map<Long, Integer> productCountMap = batchCountProducts(brandIds);

    List<BrandVO> voList = result.getRecords().stream().map(brand -> {
        BrandVO vo = assembler.toVO(brand);
        vo.setProductCount(productCountMap.getOrDefault(brand.getId(), 0));
        return vo;
    }).collect(Collectors.toList());
    // ... 构建返回
}

private Map<Long, Integer> batchCountProducts(List<Long> brandIds) {
    if (brandIds.isEmpty()) return Map.of();
    LambdaQueryWrapper<Product> wrapper = new LambdaQueryWrapper<Product>()
            .in(Product::getBrandId, brandIds)
            .select(Product::getBrandId);
    List<Product> products = productMapper.selectList(wrapper);
    return products.stream()
            .collect(Collectors.groupingBy(Product::getBrandId, Collectors.counting()))
            .entrySet().stream()
            .collect(Collectors.toMap(Map.Entry::getKey, e -> e.getValue().intValue()));
}
```

```java
@Override
@Transactional
public void batchShip(BatchShipDTO dto) {
    // 1. 数量限制
    if (dto.getOrderIds().size() > 100) {
        throw new BusinessException(400, "单次批量发货最多100单");
    }

    // 2. 查询待发货订单
    LambdaQueryWrapper<Order> wrapper = new LambdaQueryWrapper<Order>()
            .in(Order::getId, dto.getOrderIds())
            .eq(Order::getStatus, "PAID");
    List<Order> orders = list(wrapper);

    // 3. 校验是否全部匹配（防止ID不存在或状态不对）
    if (orders.size() != dto.getOrderIds().size()) {
        Set<Long> foundIds = orders.stream().map(Order::getId).collect(Collectors.toSet());
        List<Long> missingIds = dto.getOrderIds().stream()
                .filter(id -> !foundIds.contains(id))
                .collect(Collectors.toList());
        throw new BusinessException(400, "以下订单不存在或状态不允许发货: " + missingIds);
    }
```

| [AdminAuthController.java#L36-42](d:\Java_study\shop_v3\GoBuy\src\main\java\com\example\gobuy\modules\admin\controller\AdminAuthController.java#L36-42) | `X-Forwarded-For` 可被伪造，未校验是否来自真实代理 |

**修复方案**：

```java
private String getClientIp(HttpServletRequest request) {
    String xForwardedFor = request.getHeader("X-Forwarded-For");
    if (StringUtils.hasText(xForwardedFor) && !xForwardedFor.startsWith("10.") && !xForwardedFor.startsWith("192.")) {
        // 只取第一个IP（真实用户IP），忽略伪造
        return xForwardedFor.split(",")[0].trim();
    }
    return request.getRemoteAddr();
}
```

```java
private Map<Long, UserStats> getUserStatsMap(List<User> users) {
    List<Long> userIds = users.stream().map(User::getId).collect(Collectors.toList());
    if (userIds.isEmpty()) return Map.of();

    LambdaQueryWrapper<Order> wrapper = new LambdaQueryWrapper<Order>()
            .in(Order::getUserId, userIds)
            .eq(Order::getStatus, "COMPLETED");
    List<Order> orders = orderMapper.selectList(wrapper);

    return orders.stream().collect(Collectors.groupingBy(Order::getUserId,
            Collector.of(
                    () -> new UserStats(),
                    (s, o) -> { s.count++; s.totalAmount = s.totalAmount.add(o.getTotalAmount()); },
                    (s1, s2) -> { s1.count += s2.count; s1.totalAmount = s1.totalAmount.add(s2.totalAmount); }
            )));
}

@Data
private static class UserStats {
    int count = 0;
    BigDecimal totalAmount = BigDecimal.ZERO;
}
```

## 四、建议改进项

- 品牌名称映射 `getBrandNameMap()` 可以加缓存（Redis/本地），避免每次查询全表

### 4.2 异步处理
- `AdminOperationLogServiceImpl` 已使用 `@Async`，其他耗时操作（如批量发货通知）可考虑异步

### 4.3 接口限流
- 登录接口建议接入限流组件，防止暴力破解
- 批量操作接口建议限流

### 4.4 审计字段完善
